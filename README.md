# 🐾 Refugio Huellitas - Django Web App

Proyecto de sitio web multipágina desarrollado con el framework **Django**, aplicando el patrón de arquitectura **MVT (Model-View-Template)**, modularización de plantillas y buenas prácticas de desarrollo frontend (HTML, CSS y JavaScript).

Este proyecto fue realizado como entrega práctica para la **Diplomatura en Programación**.

---

## 🚀 Características del Proyecto

* **5 Secciones dinámicas:** Inicio, Mascotas, Cómo Ayudar, Nosotros y Adopciones (Contacto).
* **Modularización de código (DRY):** Uso de herencia de plantillas (`{% extends %}`) y fragmentos reutilizables (`{% include %}`) para la barra de navegación y el pie de página.
* **Interactividad (JavaScript):**
    * Filtro dinámico de mascotas (perros/gatos) en tiempo real sin recargar la página.
    * Validación interactiva y alertas de éxito/error en el formulario de adopción.
* **Diseño Custom (CSS):** Interfaz amigable, moderna, limpia y adaptativa (responsive) para dispositivos móviles y computadoras.
* **Código Limpio:** Todo el desarrollo se encuentra debidamente comentado explicando el flujo de Django y la lógica de negocio.

---

## 🛠️ Tecnologías utilizadas

* **Backend:** Python 3 & Django Framework
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
